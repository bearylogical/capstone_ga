Background
There is a widely held claim that political apathy is rife in Singapore.
Other Interests?
Under a largely one party rule for the last 55 years, disinterest in the Singapore parliamentary system does not come at a surprise considering the largely deterministic nature of debates that have occured.
Renewing Interest
Attempts to improve interest in the ongoings of the Singapore parliament have been largely met with confused responses.

The latest attempt - Parley the lion mascot - has been met with a largely puzzled response in the online community.
A more "modern" Parliament?
Fast parliament is a forward-looking concept to generate high-quality data on Singapore's Parliament via the Hansard Database.

Using Natural Language Processing (NLP) techniques on a ten year corpus (2009-2019), FastParliament address 3 key concerns.
Problem One
Parliamentary Speeches can be hard to follow. Moreover, the total time parliamentarians spent talking have been increasing each year.
Problem Two
Each speech is a treasure trove of information.

How do we distill the information such that we only get key points?
Problem Three
How can we best understand each parliamentarian?

More importantly, how should we start looking at relationships within the Hansard?
